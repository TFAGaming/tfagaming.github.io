About us:

SimplifiedCaptcha is a Discord bot that protects any Discord server from raids from generated hacked accounts. It generates random captcha codes, so it's going to be 0.0001% of chance from one of the generated hacked accounts to get the correct captcha code. Our bot includes these features below:

  1. Hosted 24/7: The bot is always online, sometimes shuts down when the monitor is timeouted.
  2. Verification: The bot includes the verification system, it generates random captcha codes.
  3. Auto Moderation: Our client is having the Anti-swear module, you can enable it.
  4. Ghost-ping Detector: You can enable this module so you can catch automatically the Ghost pingers.
  5. Open source: Going to be added soon...
Some features could be added in the future.

How it works:

SimplifiedCaptcha's verification system generates a random Captcha codes when someone requests the verified role.
If an unverified user used the command sc!verify, a text channel will be created for them and they start doing the Captcha verification.
If the user didn't provided the correct captcha code, the channel will be automatically deleted in few seconds. Else, if the user didn't provided provided the captcha code in the requested captcha code time, the channel will be automatically deleted. Else, if the captcha code is correct, the bot will add to the user the Verified role and the bot welcomes the user!
